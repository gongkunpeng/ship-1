使用Ship;)

使用Ship, 快上船!

ship init

使用ship init sitename创建一个ship站点.
ship会在当前目录下创建一个名为sitename的目录, 这个目录是ship的工作目录, 然后从github上clone主题, 默认主题是ship-theme-cat. 最后, 启动dev server, 你便可以访问http://127.0.0.1:5050, 看到可爱的site啦.

配置ship站点

ship站点的配置文件在{site}/config.py; 分为3类配置: ship基础配置Config, 示例配置ExampleConfig和属于用户的配置MyConfig; 配置项详解:
Config类

  1. DEBUG=True: 默认开启dev server的调试功能, 开启后可显示详细报错信息
  2. FLATPAGES_AUTO_RELOAD = DEBUG: 自动reload pages下的文件, 方便本地预览
  3. FLATPAGES_EXTENSION = '.md': pages下存储文件的后缀名, 默认是markdown文件


ExampleConfig类

  1. [site setting]: ship站点相关配置
    • SITE_NAME = "shipsite": 站点名称
    • SITE_URL = "https://neo1218.github.io": 站点网址, 同时也是static的根地址(必须)
    • SITE_DESC = "a ship site": 站点描述
    • SITE_OWNER = "neo1218": 站点作者
    • SITE_KEYWORDS = "ship neo1218": 站点关键字
  2. [article setting]: pages文章配置
    • ARTICLE_PER_PAGE = 10: 分页, 值为0表示禁止分页
  3. [owner info]: 站点作者信息
    • GITHUB_URL = "https://github.com/neo1218": github
    • WEIBO_URL = "http://www.weibo.com/5551886705/profile": 微博
    • TWITTER_URL = "https://twitter.com/neo1218substack": Twitter
    • QQ = "834597629": QQ
    • EMAIL = "neo1218@yeah.net": Email
  4. [deploy on github/(git) pages]: 静态站点git部署配置
    • GIT_URL = "https://github.com/neo1218/ship": git仓库url
    • BRANCH = "gh-pages": 部署的git分支

ship new

配置好FLATPAGES_EXTENSION后就可以使用ship new生成相应后缀名的文件, 比如

$ship new test

会在{site}/app/pages 下生成test.md文件, ship new会生成以下字段

title: 标题
date: year-mm-dd h:m:s (自动生成)
tags: ['标签1', '标签2']

现在你可以放开脑洞, 编写你的文章了!

ship build

ship的核心是提供一个静态化工具, 使用ship build可以静态化你的本地站点{site}到{site}/app/build目录下. 你可以将build目录部署到服务器上, 实现无数据库存储.

ship upload

ship提供了ship upload命令, 方便使用者进行git化部署. 首先你需要配置config文件中的[deploy on github/(git) pages], 然后运行git upload, 这样你的静态化目录build中的内容就部署到了git pages上. 本地仓库则位于{site}/.harbor. 最好不要直接在.harbor中进行git操作, 避免造成git冲突, 导致ship upload出错.

ship upgrade

ship upgrade命令帮助你快速的切换主题, 你只要将相应的主题clone到{site}/app/themes/目录下, 然后运行ship upgrade <themename>即可, 例如:

$ cd {site}
$ git clone https://github.com/neo1218/ship-theme-cat app/themes/cat
$ ship upgrade cat

就可以使用ship-theme-cat主题!
如果你希望编写ship主题: 详见: Theme Writer:).

ship server

启动开发服务器, 进行本地预览.
