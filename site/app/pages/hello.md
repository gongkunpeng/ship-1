How 2 use l generate static site
===

### Install
<code>$ pip install l</code>

### Init
<code>$ l init blog</code>

### Config
blog/config.py

### Run
<code>$ l server</code>

### Build
<code>$ l build</code>

### Upload
<code>$ l upload</code>
