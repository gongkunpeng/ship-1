安装Ship

0. 安装Python环境

ship是用python2编写的, 目前不能用python3运行

  1. *nix系统和Mac默认都安装了Python2
  2. Windows用户详见:
    • Windows安装Python2

1. 安装pip

pip是python的包管理器, 用pip安装ship

  1. Windows
    • 详见pip-and-setuptools-on-windows
  2. Mac
    • $sudo easy_install pip
  3. Linux
    • $apt-get install python-pip

3. 使用pip安装ship

$pip install ship-cli

4. 从源码安装pip

你也可以直接从源码安装pip

$git clone https://github.com/neo1218/ship.git
$cd ship $virtualenv venv && source venv/bin/activate
$python setup.py --editable .

这样你对源码的任何改动都会在你的虚拟环境venv中表现出来;)
